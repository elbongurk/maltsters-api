# Maltsters #

Proof of concept Rails API app for retrieving malt lot analysis data from maltsters.

## To Install ##

    # Download
    git clone https://github.com/elbongurk/maltsters-api.git
    
    # Setup
    cd maltsters-ios
    ./bin/setup

    # Fetch malt and lot data
    rake fetch
    
    # Generate malt analysis data
    rake analyze:malts
    
    # Run
    rails server
